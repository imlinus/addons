## TV4 Play - Easy image and trailer extractor

This addon allows you to download high-resolution backdrops, trailers, and show logos directly from TV4 Play program pages.

### Features
- **Download Backdrop**: Extracts the high-res hero image from the program page.
- **Download Trailer**: Gets the MP4 trailer source if available.
- **Download Logo**: Finds the original show/program logo.

### How to use
1. Go to any program page on [tv4play.se](https://tv4play.se).
2. Click the extension icon in your browser toolbar.
3. Use the buttons to download the available media.
